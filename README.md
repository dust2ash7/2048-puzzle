## ✨ Features

- True sliding tile animations
- Satisfying particle bursts on merges
- Sound effects for moves, merges, spawn, win, and game over
- Win celebration with confetti-style particles
- Game over shake animation
- Beautiful dark UI with Tailwind CSS
- Fully responsive + Progressive Web App (PWA) support
- Keyboard (arrow keys) + mobile swipe controls
- Score and best score saved locally
- Clean start screen with rules

## 🎮 How to Play

Slide the tiles **up**, **down**, **left**, or **right** to combine numbers.  
When two tiles with the same number touch, they **merge** into one with their sum.  
A new tile (2 or 4) appears after every move.  
Reach **2048** to win the game!

**Controls:**
- **Desktop**: Use arrow keys
- **Mobile**: Swipe in any direction
